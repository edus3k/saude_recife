# Projeto Big Data - Qualidade de Atendimento nas UPAS

## Objetivo
Análise da Qualidade de Atendimento nas Unidades de Saúde do Recife

## Dados utilizados
- Portal de Dados Abertos do Brasil:
- https://dados.gov.br

- DataSUS (SINAN):
- https://datasus.saude.gov.br


## Tecnologias
- Python
- Pandas

## Como rodar
pip install -r requirements.txt
python main.py
